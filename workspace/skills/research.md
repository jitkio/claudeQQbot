# 调研技能（增强版）

触发方式: 用户说"帮我调研xxx"、"查一下xxx"、"搜一下xxx"

## 执行步骤

### 1. 拆解问题
将调研主题拆解为 3-5 个不同角度的搜索关键词。
- 中文关键词为主
- 必要时补充英文关键词
- 从不同维度覆盖：定义、现状、对比、评价、趋势

### 2. 并行搜索
使用 multi_search.cjs 并行搜索多个关键词：
```bash
node /home/ubuntu/claudeqqbot/tools/multi_search.cjs "关键词1" "关键词2" "关键词3"
```

### 3. 深入抓取
对重要结果用 web_fetch.cjs 抓取全文：
```bash
node /home/ubuntu/claudeqqbot/tools/web_fetch.cjs "URL" 8000
```

### 4. 综合分析
- 交叉验证信息可靠性
- 区分事实和观点
- 用自己的话总结

### 5. 输出报告格式（纯文本，不用 markdown）

调研报告: xxx

【核心发现】
1. ...
2. ...
3. ...

【详细分析】
...

【来源】
- 标题 (URL)
- 标题 (URL)

注意：
- 多角度搜索，至少 3 个不同关键词
- 优先权威来源（官方网站、学术论文、政府机构）
- 搜完用自己的话总结，不要照抄原文
- 附上来源链接方便用户验证
