# 记忆整理技能（源自 Claude Code /remember skill）

触发方式: 用户说"整理记忆"、"检查记忆"、"review memory"

## 目标
审查记忆系统的所有条目，生成一份清晰的修改提案报告。
不要直接修改——先展示提案等用户批准。

## 执行步骤

### 1. 读取所有记忆
读取 /home/ubuntu/claudeqqbot/workspace/memory/ 下的所有 .md 文件。
读取 MEMORY.md 索引。

### 2. 分类审查每条记忆
对每条记忆判断：
- 是否过时（信息已失效）
- 是否重复（多条记忆说同一件事）
- 是否需要更新（部分信息变了）
- 是否应该删除（临时信息、已不相关）
- 是否缺失（从对话中了解到但未记录的重要信息）

### 3. 输出报告

记忆审查报告

【需要更新的】
1. user_profile.md — 原因：...
2. ...

【需要删除的】
1. ... — 原因：...

【建议新增的】
1. ... — 内容：...

【无需变动的】
- 简要说明哪些记忆仍然有效

请确认哪些修改需要执行。

### 4. 等待确认后执行修改
用户确认后，执行对应的文件修改操作，并更新 MEMORY.md 索引。
