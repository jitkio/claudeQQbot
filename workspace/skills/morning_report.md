# 早报技能（增强版）

触发方式: 定时任务每天早上自动执行，或用户说"早报"、"今天有什么"

## 执行步骤

### 1. 获取信息
并行搜索多个维度：
```bash
node /home/ubuntu/claudeqqbot/tools/multi_search.cjs "今日新闻要闻" "今日天气合肥" "科技新闻今日"
```

### 2. 读取今日课程
读取 /home/ubuntu/claudeqqbot/tools/remind_class.cjs 获取课程表数据，
根据当前日期和周数计算今天有什么课。

### 3. 读取待办任务
读取 /home/ubuntu/claudeqqbot/workspace/scheduled_tasks.json 获取今天的定时任务。

### 4. 输出格式（纯文本）

早上好，拙华！

【今日课程】第X周 周X
- 课程名 | 节次 | 教室 | 教师
- 课程名 | 节次 | 教室 | 教师
（如果今天没课则提示"今天没有课，好好休息！"）

【今日要闻】
1. 标题 — 简要描述
2. 标题 — 简要描述
3. 标题 — 简要描述

【今日天气】
城市 温度 天气状况

【待办提醒】
- 任务名 (时间)

祝你今天一切顺利！
